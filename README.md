# 🇰🇷 South Korea Tech Sales: End-to-End Data Pipeline & BI Dashboard

## 🚀 Project Overview

This project simulates, processes, and visualizes regional technology product sales data across major metropolitan areas in **South Korea** (Seoul, Busan, Incheon, Daegu, Daejeon). 

It builds a **complete, automated data pipeline (ETL)** designed to translate complex, messy sales transactions into actionable business intelligence insights. The project directly demonstrates proficiency in handling complex datasets, engineering data pipelines via Python, executing advanced SQL analytical queries, and creating clear, dynamic visualizations to support sales operations—aligning with the core requirements of Apple's Data Analyst - Korea Sales role.

### 🌟 Key Features

1.  **Automated ETL Pipeline (Python):** Ingeniously generates synthetic, yet realistic, sales transaction data with embedded business logic (e.g., Seoul population weight, specific product price points, seasonal discounts).
2.  **Advanced SQL Analysis (SQL):** Processes the data within an SQLite/PostgreSQL database using complex queries, including recursive CTEs, joins, and critically, **window functions** (RANK, MOVING AVERAGE) for trend analysis.
3.  **Actionable Data Visualization (Matplotlib & Tableau):** Includes static Python visualizations (Matplotlib/Seaborn) and a link to a **live, interactive Tableau Public Dashboard** for dynamic data exploration by sales managers.
